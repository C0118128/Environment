## Getting Started / スタートガイド

"nodebrew"でバージョン管理を行い、"node"のインストールを行う。

### Prerequisites / 必要条件

- "Node"のインストール

### Installing / インストール

"Node"インストール時に自動でインストール済み。

#### Version / バージョン確認

```
npm --version
```

### How to / 使用方法

#### Flow / 流れ

```
npm init		# 初期	# 初期化
or
npm install		# クローン時	# 使用パッケージのインストール

npm install <package>
	# 
	# -g 			#globalにインストール

npm run <script>
```
